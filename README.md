# MamaPutOnline

A landing page for a Nigerian online takeaway restaurant — built with HTML, CSS, and vanilla JavaScript.

Live demo: [nnamdi-security.github.io/mamaputonline](https://nnamdi-security.github.io/mamaputonline/)

---

## About This Project

MamaPutOnline is a concept landing page designed for a Nigerian takeaway food delivery service. It was built to practice real interactive JavaScript — sliders, scroll-based animations, and hover interactions — beyond static HTML and CSS.

This project is part of my #BiP60 Build in Public challenge — 60 days of learning and building in public as a trainee in the Enugu State Government technology programme run in partnership with Turing Tech LLC.

---

## Built With

- HTML5 — semantic structure
- CSS3 — animations, transitions, responsive layout
- Vanilla JavaScript — sliders, scroll detection, event handling

---

## Features

- **Sticky navbar** with a responsive hamburger menu for mobile
- **Hero section food card slider**
  - Auto-slide
  - Dot indicators
  - Manual previous/next controls
- **Scroll reveal animations** — sections fade in as they enter the viewport, using the Intersection Observer API
- **Hover animations** on buttons and cards
- **Review slider**
  - Swipe support on touch devices
  - Pauses automatically on hover (desktop)
- **Contact form** for inquiries
- Fully responsive across mobile, tablet, and desktop

---

## Sections

- **Navbar** — sticky, with hamburger menu on mobile
- **Hero** — auto-sliding food card carousel
- **Menu** — featured dishes with hover effects
- **Reviews** — swipeable, auto-pausing testimonial slider
- **Contact** — inquiry form
- **Footer** — links and contact details

## Getting Started

To run this locally:

```bash
git clone https://github.com/nnamdi-security/mamaputonline.git
cd mamaputonline
```

Then open `index.html` in your browser, or run it with a local server (e.g. VS Code's Live Server extension) for the best experience with the JavaScript features.
