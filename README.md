# MamaPutOnline

*Authentic Nigerian meals delivered fresh to your doorstep.*

A full landing page for a Nigerian online food delivery service — built with HTML, CSS, and vanilla JavaScript.

Live demo: [mamaputafrica.netlify.app](https://mamaputafrica.netlify.app/)

---

## About This Project

MamaPutOnline is a concept food delivery website built to bring authentic, home-style Nigerian meals online — soups, rice dishes, proteins, shawarma, and combo meals — with a smooth, modern ordering experience.

Beyond the business concept, this project was built to practice real interactive JavaScript: sliders, scroll-based animations, search and filtering, and hover interactions, going well beyond static HTML and CSS.

This project is part of my #BiP60 Build in Public challenge — 60 days of learning and building in public as a trainee in the Enugu State Government technology programme run in partnership with Turing Tech LLC.

## Built With

- HTML5 — semantic structure
- CSS3 — animations, transitions, responsive layout
- Vanilla JavaScript — sliders, scroll detection, filtering, event handling
- Netlify — hosting and deployment

---

## Features

- **Sticky navbar** with search icon and a responsive hamburger menu for mobile
- **Hero section image slider** with previous/next controls showcasing signature dishes
- **Full menu system** organised into categories:
  - Soups
  - Rice & Local Dishes
  - Protein Specials
  - Shawarma & Specials
  - Combo meals (Special, Family, and Student value combos)
- Each menu item includes **image, price, description, and star rating**
- **Scroll reveal animations** — sections fade in as they enter the viewport, using the Intersection Observer API
- **Hover animations** on buttons and cards
- **Customer review carousel**
  - Swipe support on touch devices
  - Pauses automatically on hover (desktop)
  - Previous/next navigation arrows
- **Services section** highlighting fast delivery, fresh ingredients, nationwide ordering, and secure payment
- **Contact form** with phone, email, business hours, and location
- **Newsletter subscription** field in the footer
- Fully responsive across mobile, tablet, and desktop

## Sections

- **Navbar** — sticky, with search and hamburger menu on mobile
- **Hero** — image slider with signature dish highlights and call-to-action buttons
- **About (Get To Know Us)** — mission statement and reasons customers trust the brand
- **Menu (Our Menu)** — categorised dish listings with pricing and ratings
- **Services (Our Services)** — delivery, freshness, reach, and payment security
- **Reviews** — swipeable, auto-pausing testimonial carousel
- **Contact** — inquiry form and direct contact details
- **Footer** — quick links, contact info, newsletter signup, and social links

## Getting Started

To run this locally:

```bash
git clone https://github.com/nnamdi-security/mamaputonline.git
cd mamaputonline
```

Then open `index.html` in your browser, or run it with a local server (e.g. VS Code's Live Server extension) for the best experience with the JavaScript features.
