# 🍲 MamaPutOnline — Authentic Nigerian Meals Delivered Fresh

A responsive, single-page food delivery website for **MamaPutOnline**, a fictional Nigerian home-style meal delivery service. The site showcases a rich menu of authentic Nigerian dishes, highlights key services, and provides a seamless contact and ordering experience — all built with HTML and CSS.

**🔗 Live Site:** [nnamdi-security.github.io/mamaputonline](https://nnamdi-security.github.io/mamaputonline/)

## 📄 Page Sections

MamaPutOnline is a single-page application with smooth scroll navigation across six sections:

| Section | Description |
|---------|-------------|
| **Home** | Full-screen hero with image slideshow, tagline and call-to-action buttons |
| **Get To Know Us** | Brand story, mission statement and trust-building highlights |
| **Our Menu** | Tabbed menu with 5 categories and 35+ dishes, each with image, name, price and star rating |
| **Our Services** | Four service pillars — fast delivery, fresh meals, nationwide ordering, and secure payment |
| **Reviews** | Customer testimonial carousel with photos and ratings |
| **Contact Us** | Contact form, business hours, phone/email, address, and newsletter subscription |

---

## 🍽️ Menu Categories

The menu is organised into five tabbed categories:

- **Soups** — Egusi, Ewedu, Ora, Ogbono, Edikang Ikong, Afang, Bitterleaf, Nsala, Ofe Owerri, Black Soup, Efo Riro, Okro Soup
- **Rice & Local Dishes** — Jollof Rice, Fried Rice, Native Jollof, Porridge Yam, Porridge Beans, Abacha, Ukwa, Nkwobi, Moi Moi, and more
- **Protein Specials** — Goat Meat Pepper Soup, Fried Chicken, Fried Turkey, Beef & Chicken Suya, Asun, Peppered Gizzard, Grilled Catfish, Grilled Croaker
- **Shawarma & Specials** — Chicken, Beef, and Cheese variants
- **Combo Meals** — MamaPut Special Combo, Family Feast Combo, Student Value Combo

---

## ✨ Features

- Fully responsive layout for desktop, tablet, and mobile
- Hero section with auto-rotating image slideshow
- Tabbed menu navigation for easy browsing across categories
- Dish cards with food photography, pricing (in ₦), and star ratings
- Customer testimonial carousel
- Contact form with full name, email, phone, and message fields
- Newsletter subscription section in the footer
- Sticky navigation bar with hamburger menu for mobile
- Smooth scroll behaviour throughout the page
- Back-to-top button

---

## 🛠️ Built With

- **HTML5** — Semantic, single-page structure with anchor-based navigation
- **CSS3** — Responsive layout, card design, tabs, carousel, and animations
- **JavaScript** — Slideshow, tab switching, testimonial carousel, and mobile menu toggle
- **GitHub Pages** — Deployment and hosting

## 🚀 Getting Started

To run the project locally:

```bash
# Clone the repository
git clone https://github.com/nnamdi-security/dev.git

# Navigate to the project folder
cd dev/mamaputonline

# Open in your browser
open index.html
```

No dependencies or build tools required.
